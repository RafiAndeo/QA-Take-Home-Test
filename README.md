# QA-Take-Home-Test
QA Take Home Test from Paper that includes Section A, Section B, and Section C
Below are the instructions for setup the environment on code tests:

# Section B

1. This code is running using Eclipse IDE, you can download the newest version by accessing this link: https://eclipseide.org/
   
2. After you have download and install the IDE, you need to download Java SDK from this link: https://www.oracle.com/asean/java/technologies/downloads/
   
3. You can make sure the Java SDK by accessing your CMD or PowerShell then type java --version
   
4. You need to download the Selenium Java library from this link: https://www.selenium.dev/downloads/
   
5. After you have installed it, you need to adjust your eclipse workspace by downloading this folder and set it as your default workspace
    
6. Next, you will need to set the Selenium Java library to this folder by click properties then access Java Build Path. You can go to Libraries then click Classpath then click the Add External JARs button. You need to select all available Selenium Java libraries then finally you can click Apply and Close
   
7. Both codes are available using Google Chrome or Microsoft Edge as the browser to test. You can download those drivers by accessing these link below (match your browser version with the drivers that you are going to use):

Chromedriver: https://googlechromelabs.github.io/chrome-for-testing/
Msedgedriver: https://developer.microsoft.com/id-id/microsoft-edge/tools/webdriver/?cs=2090951799&form=MA13LH#downloads

8. Make sure you change the path from the code to match the path where you put the downloaded drivers

9. You should be able to run the code

# Section C

1. You will need to download or install Postman to run the collection, you can access it by this link: https://www.postman.com/downloads/

2. After you have installed it, you can import this collection to your machine

3. You can continue running the collection by clicking the Send button

4. For the performance test, you will need to install Visual Studio Codeo, you can access it here: https://code.visualstudio.com/download?_exp_download=fb315fc982

5. This code use K6 JavaScript, you can see this documentation on how to install and run it: https://grafana.com/docs/k6/latest/set-up/install-k6/

6. On the code, you can change the virtual users and duration to test other scenario as you like

Thank you and happy coding
